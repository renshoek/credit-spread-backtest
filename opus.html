Parameter guide
Short OTM % — How far below current SPY the short put strike is. 5% = sell put at 95% of SPY price.
Long OTM % — How far below current SPY the long put strike is. Must be deeper than short. Defines max loss width.
DTE — Days to expiration. Standard monthly = 30. Weeklies = 7. Affects premium via time decay.
Margin use % — Fraction of total capital deployed as margin each month. 25% = professional standard. 100% = guaranteed ruin.
Short/Long skew × — IV multipliers applied to VIX. OTM puts trade at higher IV than ATM (the "skew"). 1.30/1.50 are conservative estimates for SPY. Higher = bigger premiums collected.
VIX min — Skip months where VIX is below this level. Low VIX = tiny premiums, not worth trading. Try 12–15.
Stop-loss % — Close early if unrealized loss exceeds this % of max possible loss. 0 = hold to expiry. 50 = cut losses at half.
Slippage $ — Per-spread cost deducted from premium. Accounts for bid-ask spread + commissions. Real value: $1–3.
Risk-free rate — Used in Black-Scholes pricing. "Historical" uses actual monthly Fed Funds Rate (2000–2024). "Fixed" uses your input.

⚠ THIS IS A SIMULATION, NOT A REAL BACKTEST. Option prices are computed via Black-Scholes using VIX as an IV proxy with skew adjustments. Real options have actual traded prices, bid-ask spreads, and IV surfaces that differ significantly. A real backtest requires historical options chain data (CBOE OptionMetrics, ~$10k+/yr). Premiums may be 10–30% off from real market prices.

⚠ LIMITATIONS: Monthly SPY closes only — intra-month path unknown (gap risk, early assignment). No dividend impact. No term structure. No gamma risk. Excludes IBKR commissions (~€1.30/trade) unless slippage is set. Past performance does not guarantee future results. Paper-trade before real capital.
