Parameter guide & methodology
Short OTM % — How far below current SPY the short put strike is. 5% = sell put at 95% of SPY price. Lower = more premium, higher chance of being tested.
Long OTM % — How far below current SPY the long put strike is. Must be deeper than short. Defines max loss width (spread width = long − short).
DTE — Days to expiration. Standard monthly = 30, Tastytrade = 45. Outcome is still checked at month-end close.
Margin use % — Fraction of capital deployed as margin each month. 25% = professional standard. At 25%, a full-loss month costs −25% of capital.
Short/Long skew × (Simulated mode only) — Static IV multipliers applied to VIX. 1.30/1.50 are estimates for 5%/8% OTM puts. In Real mode these are replaced by dynamic SKEW-derived values per month.
CBOE SKEW Index (Real mode) — Measures cost of tail-risk protection. Formula: mult = 1 + (SKEW−100) × OTM% / 500. SKEW=130 at 5% OTM gives ×1.30. Steepens in crashes, flattens in calm markets.
Intra-month VIX high (Real mode) — Uses actual daily VIX peak within each month for path estimation, not just month-end close. Oct 2008: close=59.9 but intra-month high=89.5.
VIX floor / ceiling — Skip months if VIX outside range. 0 / 999 = always trade.
Stop-loss × prem — Close early if loss ≥ this multiple of premium. 0 = hold to expiry. Tastytrade standard: 2×.
Slippage $ — Per-spread cost deducted from premium (bid-ask + commissions). Real: $1.50–3.00.
Risk-free rate — Historical = actual monthly Fed Funds Rate (FRED). Fixed = your input.
